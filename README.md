# Python Pomodoro Timer 🍅

[English]
This is my first Python project, developed to practice Object-Oriented Programming (OOP) concepts and terminal-based user interaction. It helps users manage their time effectively using the Pomodoro technique.

[Türkçe]
Bu, nesne yönelimli programlama (OOP) kavramlarını pekiştirmek ve terminal tabanlı kullanıcı etkileşimini öğrenmek için geliştirdiğim ilk Python projemdir. Pomodoro tekniğini kullanarak kullanıcıların zamanlarını verimli yönetmelerine yardımcı olur.

## Features / Özellikler
- **OOP Design:** Built using classes for a modular structure. / Modüler bir yapı için sınıflar (class) kullanılarak tasarlandı.
- **Error Handling:** Robust input validation using `try-except` blocks. / `try-except` blokları kullanılarak hatalı girişler engellendi.
- **Terminal UI:** Clean and interactive countdown display. / Sade ve etkileşimli bir geri sayım ekranı.

2. Clone the repo or download the file. / Depoyu klonlayın veya dosyayı indirin.
